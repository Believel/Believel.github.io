搭建个人博客专栏

Veröffentlicht am 2015-10-07

基于git搭建个人博客的专栏

–在github上的准备工作

1.首先创建一个新的仓储，仓储的名字：github名+ github.io
2.克隆到本地git clone git地址 --depth 1

–安装基于node开发的静态博客生成器hexo

  1. hexo官网
  2. npm install -g hexo-cli
  3. 安装一些依赖之后，初始化项目hexo init blog
  4. 启动服务hexo server这样就可以在本地中浏览项目
  5. 创建编写笔记的文件hexo new 文件名字
  6. 这是基于markdown编写的，要想在网页中查看，还要生成静态页面hexo generate这是会生成一个public文件夹

  7. 为了发布方便，下面就需要完成自动发布的流程
    找到_config.yml文件中的Deploy属性指定

  8. 完成自动部署
    npm install hexo-deployer-git --save
    hexo deploy --generate
    这样就可以了
